# Proppd

Modern South African property portal.

## MVP

Portal-first platform for:

- property search
- listing pages
- agent and agency profiles
- enquiry capture
- admin listing management
- agent lead management

## Product Docs

- Product brief: `docs/product-brief.md`
- MVP implementation plan: `docs/plans/2026-05-18-proppd-mvp.md`

## Security Note

Do not commit cPanel, Supabase, email provider, or hosting credentials. Use `.env.local` for local secrets and `.env.example` for placeholders only.
